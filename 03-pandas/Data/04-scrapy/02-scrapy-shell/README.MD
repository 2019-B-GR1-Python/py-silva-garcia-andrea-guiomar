# Scrapy

## Scrapy instalacion

Ejecutar dentro del `anaconda prompt`

```
$pip instal scrapy

# Comando generales
```
$scrapy bench

Da las caracteristicas para poder hacer Web Scraping o web Crawling de ese computador
``````

```
$scrapy settings

visualizar configuraciones extras
``````

```
$scrapy version

visualizar configuraciones extras
``````

### scrapy view `url`

``````
$scrapy view https://www.pluralsight.com/authors
``````
